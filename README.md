## 使用方法

### 方法一：直接编辑HTML
1. 用任意文本编辑器打开 .html 文件
2. 替换示例内容为你的信息
3. 在浏览器中打开预览
4. 浏览器打印 → 另存为PDF

### 方法二：批量转PDF
```bash
# 安装转换工具（仅需一次）
brew install wkhtmltopdf

# 运行转换脚本
python3 convert_to_pdf.py
```

### 方法三：在线编辑器
1. 打开HTML文件
2. 右键 → 用浏览器打开
3. 按F12打开开发者工具
4. 使用"元素选择"工具直接点击修改内容
5. 修改后打印保存

## 修改内容

所有模板使用纯HTML+CSS，无需编程知识：

- **个人信息**：修改 `.name` `.title` `.contact` 等标签内的文字
- **工作经历**：复制 `<div class="item">` 块添加新经历
- **技能标签**：修改 `<span class="skill-tag">` 内的文字
- **颜色**：修改CSS中的颜色值即可换配色
